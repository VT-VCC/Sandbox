Version 1.0

Features:
-SD Logging for Ground Station 2
-Beacons TX at 5 second interval with +-2.5 sec randomization
-Digi beacon locations at same interval and randomization as beacons
-Packet filtering based on RFM69HCW features (NETID and NODEID)
-Ground Station in 'filtering' mode
-Path: Beacons -> Quad -> Airplane -> Ground Station
-Tested with two beacons, two digis, one ground station
-Full power on all devices
-Frequency Calibration of all devices (ensure close to same freq for all nodes).
